Un serveur HTTP simple codé en C++

Jeancerver 1.0 est un serveur basé sur le protocole HTTP 1.1 dit "Hypertext Transfer Protocol" qui permet de transmettre des documents d'un serveur vers un client comme un navigateur web.

Son principe est simple. Le client ouvre une connexion et envoie une requête HTTP. Le serveur traite cette requête et renvoie le fichier correspondant sous forme d'une réponse HTTP. Dans le cas d'un navigateur web, cette réponse sera alors traitée pour afficher son contenu à l'utilisateur.
